2 变量

2.1 介绍

变量存在的意义：方便管理内存空间

变量创建的语法：数据类型 变量命 = 变量初始值;

2.2 常量

常量的定义方式

  • #define 宏常量
  • const 修饰的变量

2.3 关键字

创建变量： 数据类型 变量名称 = 变量初始值
不要用关键字给变量或者常量起名称

2.4 标识符

标识符命名规则

  • 不可以是关键字
  • 标识符是由字母，数字，下划线构成
  • 标识符第一个字符只能是字母或下划线
  • 标识符是区分大小写的

3 数据类型

3.1 大纲

类型 含义 最小尺寸 所占字节
bool 布尔类型 未定义 1字节
char 字符 8位 1字节
wchar_t 宽字符 16位 -字节
char_16 Unicode字符 16位 -字节
char_32 Unicode字符 32位 -字节
short 短整型 16位 2字节
int 整形 16位 4字节
long 长整型 32位 4字节
long long 长整型 64位 8字节
float 单精度浮点数 6位有效数字 4字节
double 双精度浮点数 10位有效数字 8字节
long double 扩展精度浮点数 10位有效数字 -字节

3.2 整形

类型 short int long long long
含义 短整型 整形 长整型 长整型

3.3 浮点型

  • 默认情况下输出一个小数，显示6位有效数字
  • 科学计数法：float f2 = 3e2; 3*10^2;

3.4 字符型

字符型变量并不是把字符本身放到内存中存储，而是将相应的ASCLL编码放入到存储单元,如：
A – 65,
a – 97
